# Frontend

Dokumentasi untuk aplikasi frontend Portal Akademik.

## Variabel Lingkungan

Variabel lingkungan berikut diperlukan untuk menjalankan aplikasi. Salin file `.env.example` ke `.env` dan isi nilainya.

| Variabel | Deskripsi | Contoh |
| --- | --- | --- |
| `VITE_API_URL` | URL dasar untuk API backend. | `http://localhost:3000/api/v1` |
| `VITE_APP_NAME` | Nama aplikasi yang ditampilkan di judul. | `"Portal Akademik"` |
